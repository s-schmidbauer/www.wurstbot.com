IIS redirect HTTP to HTTPS

In case you want to redirect your unencrypted HTTP requests on your IIS to HTTPS, use this snippet in your Redirects.config file within your web root.

<rule name="redirect-to-https" patternSyntax="ECMAScript" stopProcessing="true">
    <match url="(.*)" />
    <conditions>
        <add input="{HTTPS}" pattern="^OFF$" />
    </conditions>
    <action type="Redirect" url="https://{HTTP_HOST}/{R:1}" appendQueryString="true" />
</rule>

It will redirect all incoming requests to when the protocol HTTPS is off. All query strings will be appended to the URL. Make sure you have a valid certificate in place.

You can also create such redirects using the IIS GUI.

And watch out for redirect loops! :-)
