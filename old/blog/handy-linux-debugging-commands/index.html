Handy linux debugging commands

I'll post some handy commands from time to time that I'll use over and over to determine issues.

Remeber kids, always check the logs first :-)

Show processes running, sorted by default by CPU usage

top

Check for free disk space

df -h

Check used disk space for all folders in 'myfolder'

du -sh myfolder/*

Summarized file usage of current folder

du -sSh *

List all active java processes being executed by all users

sudo ps -aux | grep java

List open files for process with ID 1234

lsof -p 1234

List open files and their network sockets for "myuser" 

lsof -i -u myuser

Search through files recursively

grep -r /home/me *.pdf
