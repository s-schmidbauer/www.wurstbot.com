About wurstbot

wurstbot.com is a dynamic web application developed in Python and the Django web-framework.

It's main goal is to provide content that is meant to be clear and simple to understand. A friction-less (fast, secure, simple) experience for it's users is it's highest priority.

The site runs on FreeBSD with apache configured with mod_wsgi. It's host is a virtual machine provided by vds6.net which offers VPS in a number of locations at very low cost.

wurstbot.com is online since January 2017
